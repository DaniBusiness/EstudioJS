# TEMARIO
#
# Este temario no es exhaustivo, en la medida en la que avancemos, vamos a parar más en algunos detalles
y conceptos. Eso dependerá del ritmo de estudio, del nivel de dificultad de algunos temas, o de contenido
relacionado que no haya contemplado inicialmente y que convenga aprender.

MÓDULO 0: Preparativos

    > Configuraciones previas

        Instalación de un editor de código (Visual Studio Code o similar)
        Configuración de un repositorio local (Git)
        Configuración de repositorio remoto (GitHub)

MÓDULO 1: Introducción a JavaScript
Objetivo: Familiarizarse con los conceptos básicos y el entorno de desarrollo.

    > Conceptos básicos de programación y configuraciones previas

        Primer programa en JavaScript: "Hello, World!"
        Sintaxis básica

        Variables y tipos de datos
        Operadores básicos
        Comentarios en JavaScript
        Ejercicios prácticos

        Escribir y ejecutar programas simples
        Ejercicios con variables y operadores

MÓDULO 2: Control de flujo
Objetivo: Aprender a controlar el flujo de los programas.

    > Condicionales

        if, else if, else
        Operadores lógicos y de comparación
        Ejercicios prácticos

    > Bucles

        for, while, do...while
        Bucles anidados
        Ejercicios prácticos

MÓDULO 3: Funciones y alcance
Objetivo: Entender el concepto de funciones y su alcance.

    > Funciones

        Declaración y ejecución de funciones
        Parámetros y argumentos
        Valor de retorno
        Funciones anónimas y funciones flecha
        
    > Alcance

        Alcance global y local
        Variables var, let y const
        Hoisting
        Ejercicios prácticos

MÓDULO 4: Arreglos y objetos
Objetivo: Manejar colecciones de datos.

    > Arreglos

        Declaración y manipulación de arreglos
        Métodos básicos de arreglos (push, pop, shift, unshift)
        Iterar sobre arreglos
    
    > Objetos

        Declaración y manipulación de objetos
        Acceso a propiedades y métodos
        Ejercicios prácticos

MÓDULO 5: DOM y eventos
Objetivo: Interactuar con el DOM y manejar eventos.

    > DOM

        Qué es el DOM
        Seleccionar elementos del DOM (getElementById, querySelector)
        Manipular el DOM (cambiar contenido y estilos)
    
    > Eventos

        Qué son los eventos
        Escuchar eventos (addEventListener)
        Ejercicios prácticos

MÓDULO 6: Asincronía y promesas
Objetivo: Trabajar con código asincrónico.

    > Callbacks y promesas

        Qué es la asincronía
        Callbacks
        Promesas (then, catch, finally)
    
    > Async/await

        Sintaxis async y await
        Manejo de errores
        Ejercicios prácticos

## Lo ideal es que, al terminar estos temas, podamos realizar un proyecto que los abarque.
